## 🎬 Demo 视频脚本与讲解大纲（建议 4–6 分钟）

**目标**：用声音讲解，覆盖核心模块，突出创新点。建议 Chrome、提前在 Vercel 配好 `FREE_LLM_* / XIAOMI_TTS_API_KEY / SILICONFLOW_ASR_API_KEY`。

**0:00–0:30 开场 / 定位**

-   一句话介绍："AI 英语口语陪练——选真实场景，开口对话，实时纠错，课后给可量化报告。纯前端 Next.js，已部署上线。"
-   亮点预告："三大创新：① 语音三段(ASR/LLM/TTS)全可配的三种模式；② 零成本即开即用；③ 可解释的发音评测 + 自动课后报告。"

**0:30–1:30 免费模式（零配置即用）**

-   首页展示场景卡片 + 顶部「免费模式」徽标。讲："不填任何 Key，浏览器语音 + 服务端内置智谱对话，直接能练。"
-   进「酒店入住」场景 → 点麦克风说一句（故意带个小错，如 "I have a repetition"）→ 展示实时纠错气泡（repetition→reservation）。
-   讲点：双阶段纠错——对话中轻提示，不打断。

**1:30–2:45 标准 / 高级模式（三段可配，核心创新）**

-   进设置页讲三种模式表：免费 / 标准（内置或自配 LLM + 内置小米 TTS）/ 高级（云端硅基 ASR + LLM + 云端小米 TTS，且都可自填自己的 API）。
-   切到「高级模式」→ 试听小米音色（自然度对比浏览器 TTS）→ 选云端识别。
-   讲点：一套代码适配多家、密钥全走服务端不泄露；高级模式录音整段识别、不被停顿截断，Edge 也能用。

**2:45–3:45 一次完整高级对话**

-   选「求职面试」→ 多轮对话：录音说一段（中间停顿）→ 云端识别准确转写 → AI 自然语音回复 → 出现纠错。
-   讲点：端到端流畅、延迟可接受、发音更自然。

**3:45–4:45 课后报告 + 学习记录（可量化反馈）**

-   结束练习 → 自动生成六维雷达图报告（发音/语法/词汇/流利度/自然度/任务完成）。
-   强调："发音/流利度基于真实信号——识别置信度与语速 WPM，算法透明、敢讲依据，不是模型臆造。"
-   进「学习记录」→ 成绩趋势折线 + 高频纠错聚合。讲点：练习结束即自动生成报告，趋势随时完整。

**4:45–5:30 工程与收尾**

-   快速带过：GitHub 多 PR/细粒度提交、vitest 单测、Vercel 持续部署、断点续接（练习中途离开可恢复）。
-   收尾一句价值主张 + GitHub 链接（右上角图标）。
